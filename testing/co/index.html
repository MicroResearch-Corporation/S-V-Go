Framework usage

    
Include src/user.js on your site to auto-inject icons.


    
Inline class mode

    
<svg class="svgo svgo-abacus"></svg>


    
Symbol shim mode (local)

    
Because source SVGs are path-only, user.js creates local symbols at runtime:

    
<svg width="24" height="24" fill="currentColor">
  <use href="#svgo-abacus"></use>
</svg>

    
First reference to an icon registers #svgo-* in a hidden sprite. Subsequent <use> are instant.


    
Dark/Light mode

    
Toggle the theme with the button in the header. The gallery uses CSS variables and prefers-color-scheme.
